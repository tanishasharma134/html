Header Tags

Understanding Header Tags

Header tags are used to define headings in an HTML document. They range from <h1> to <h6>, with <h1> being the highest level and <h6> the lowest. Proper use of header tags helps to structure content hierarchically and improves accessibility.

  • <h1> - Main heading
  • <h2> - Subheading
  • <h3> - Section heading
  • <h4> - Subsection heading
  • <h5> - Minor heading
  • <h6> - Least important heading
